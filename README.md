# 🩺 Vedansh Medical Coding Academy

**Vedansh Academy** is a modern, responsive website built to showcase professional **Medical Coding & Billing** courses.  
This project demonstrates clean design, fast performance, and mobile-first structure — ideal for institutes, academies, or coaching centers.

---

## 🚀 Live Demo  
👉 [https://vedansh-academy.vercel.app](https://vedansh-academy.vercel.app)

---

## 🎯 Features

- Responsive & modern layout  
- Dedicated sections for:
  - Course listings  
  - Key benefits (“Why Us”)  
  - Registration form (Google Form embedded)  
- Gradient design with glass-card elements  
- Smooth animations and mobile-optimized components  
- Works 100% static — **no backend required**

---

## 🧠 Tech Stack

| Technology | Purpose |
|-------------|----------|
| **HTML5** | Structure |
| **CSS3 (Custom + Gradient Design)** | Styling |
| **JavaScript (Optional)** | Menu toggle, animations |
| **Google Forms Embed** | Registration form |
| **Vercel Hosting** | Free, fast global deployment |
